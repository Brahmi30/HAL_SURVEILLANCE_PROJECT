# Real-Time AI Surveillance System for Face and Aircraft Recognition

## Project Overview
This project is a Real-Time AI Surveillance System developed using Artificial Intelligence and Computer Vision technologies. The system continuously monitors live video streams and performs real-time face recognition and aircraft detection using YOLOv8 and DeepFace models.

The project also includes a Flask-based surveillance dashboard for live monitoring, file upload recognition, downloadable logs, and real-time detection visualization.

----------------

## Features

- Real-time face recognition
- Real-time aircraft detection
- YOLOv8 object detection
- DeepFace-based facial recognition
- Flask surveillance dashboard
- Live webcam monitoring
- File upload and recognition feature
- Downloadable surveillance logs
- Multi-threaded real-time processing
- Continuous monitoring system

## AI Models Used

Object Detection -> YOLOv8 
 Face Recognition (Upload Feature) -> ArcFace 
 Face Recognition (Live Surveillance) -> GhostFaceNet 
 Aircraft Recognition -> Facenet 
 Face Detection Backend -> OpenCV
